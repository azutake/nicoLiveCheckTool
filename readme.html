ニコ生放送チェックツール

ニコ生の公式放送以外の放送の開始をチェックするツールです。

使い方

最初にツール→オプションメニューからログイン設定をします。
左上の「新規登録」ボタンからコミュニティ/チャンネルID、ユーザーID、キーワードの一つ以上を設定して登録します。

機能説明

機能についての説明です。

アラート情報の取得元

チャンネルから放送されていない公式放送は取得できません。
  • RSS
    • フォローしていないコミュニティ・チャンネル・ユーザーの放送の開始も取得することができますが、放送開始から1～3分ほど遅れることが多いです。お気に入りリストにユーザーIDが指定されていた場合には正確にはユーザー名と照合しています。設定で指定した時間ごとにユーザー名を更新していますが、放送者のユーザー名が変わっていた場合は取得漏れが発生する可能性があります。
  • ブラウザプッシュ通知
    • フォローしているコミュニティ・チャンネル・ユーザーの放送を取得することができます。リストに登録されていてもフォローしていない場合は放送開始の通知を取得できません。取得タイミングはずれることがあるようですが、2019年3月10日現在は放送開始後30秒ほどで取得できることが多いです。
  • スマホアプリプッシュ通知
    • フォローしているコミュニティ・チャンネルの放送を取得することができます。リストに登録されていてもフォローしていない場合は放送開始の通知を取得できません。取得タイミングはずれることがあるようですが、2019年3月10日現在は放送開始後数秒で取得できることが多いです。
